# Images → PDF

Script : `script.py`

Charge toutes les images `.jpg`, `.jpeg` ou `.png` présentes dans le dossier du
script, les convertit en RGB si besoin puis les assemble dans un PDF
`fusion_images.pdf`.

## Prérequis
- Python 3
- `Pillow`

Installation :
```bash
pip install pillow
```

## Utilisation
```bash
python script.py
```
- Placez vos images dans ce dossier avant de lancer le script.
- Le PDF généré est créé à côté du script.
