# Images vers PDF

Script : `script.py`

Convertit toutes les images (`.jpg`, `.jpeg`, `.png`) du dossier en un unique
fichier PDF nommé `fusion_images.pdf` en respectant l'ordre alphabétique.

## Prérequis
- Python 3
- Bibliothèque `Pillow` (`PIL`)

## Utilisation
```bash
python script.py
```
Le PDF généré est placé dans le même dossier que les images sources.
