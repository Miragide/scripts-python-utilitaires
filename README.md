# Scripts Python utilitaires

Chaque sous-dossier contient un script autonome (ou une paire de scripts)
dédié à une tâche de conversion ou de traitement. Reportez-vous aux README de
chaque répertoire pour les dépendances et instructions d'exécution.

- `aborescence maker/` : affichage d'une arborescence de dossiers.
- `fusionpdf/` : fusion de PDF en un seul fichier.
- `image to pdf/` : regroupement d'images en PDF.
- `image to txt/` : extraction OCR avec EasyOCR.
- `mp4 to mp3/` : conversion audio depuis des vidéos MP4.
- `optimages 2000 px compress/` : compression et renommage d'images.
- `pdf to img/` : conversion de pages PDF en PNG.
- `pdf to markdown api gpt/` : conversion de PDF en Markdown via OpenAI.
- `remove all img bg/` : suppression d'arrière-plan d'images.
- `transcription mp3-mp4 to txt/` : transcription Whisper d'audios/vidéos.
- `vcard to csv/` : export de contacts vCard en CSV.
- `verification numero secu/` : contrôle de NIR dans un CSV.
- `video montage photos to video/` : recadrage et concaténation vidéo carrée.
