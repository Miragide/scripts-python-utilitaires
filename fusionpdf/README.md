# Fusion PDF

Script : `script.py`

Fusionne tous les fichiers PDF présents dans le dossier courant en un seul
fichier `fusion.pdf`, après les avoir triés par ordre alphabétique.

## Prérequis
- Python 3
- Bibliothèque `PyPDF2`

## Utilisation
Depuis le dossier, lancez :
```bash
python script.py
```
Le fichier `fusion.pdf` est généré à côté des sources.
