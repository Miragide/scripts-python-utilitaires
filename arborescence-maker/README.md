# Arborescence Maker

Script : `script.py`

Ouvre une boîte de dialogue Tkinter pour choisir un dossier, puis affiche dans
le terminal toute l'arborescence en utilisant des branches ASCII.

## Prérequis
- Python 3
- Tkinter (intégré à la plupart des distributions Python)

## Utilisation
1. Placez-vous dans le dossier du script.
2. Lancez :
   ```bash
   python script.py
   ```
3. Sélectionnez le dossier cible dans la boîte de dialogue. La structure est
   imprimée immédiatement dans le terminal avec indentation et symboles.
