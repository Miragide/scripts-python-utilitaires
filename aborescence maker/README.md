# Arborescence Maker

Script : `script.py`

Cet utilitaire ouvre une boîte de dialogue graphique pour sélectionner un dossier
puis affiche dans la console l'arborescence complète avec des branches ASCII.

## Prérequis
- Python 3
- Tkinter (intégré à la plupart des distributions Python)

## Utilisation
1. Placez-vous dans le dossier du script.
2. Exécutez :
   ```bash
   python script.py
   ```
3. Choisissez le dossier à inspecter dans la boîte de dialogue. La structure sera
   imprimée immédiatement dans le terminal.
